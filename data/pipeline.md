## How to Use This File

1. **Review URLs** — click through the priority 1 and 2 postings first
2. **Remove any you don't want to apply to** — delete lines/sections
3. **Add manual finds** — paste URLs from OpenAI/DeepMind/Mistral/etc. as you scan them manually
4. **Run `/career-ops pipeline`** — batch-evaluates each URL (Blocks A-G scoring)
5. **Only 4.0+ scores surface for action** — these become real application candidates
6. **For each 4.0+ match, run `/career-center tailor-cv`** to generate company-specific variant
7. **Apply manually** via the company's portal (career-ops never auto-submits)
